Angry.g0v.tw - 怒政事件記錄簿
==========================
那些年，我們一起憤怒的政客、政策、政績、政治事件
---------------------------------------

Why
---
希望能夠一眼看出，我們的政府作錯過什麼事情。

What
----
 - 這個網站是一個以**時間**為基礎，紀錄事件的導覽網站。
 - 事件的範疇，限制在「讓人憤怒的政客、政策、政績、政治事件」。
 - 事件的來源**暫時**限定在 zh.wikipedia.org 的相關詞彙
